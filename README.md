# Apple LP 模写

## 概要
ブレイクポイントが多いWebサイトをスピーディーに作成するための練習として制作。<br>
画像・フォントファミリーは [『Apple（日本）』](https://apple.com/jp/)から引用。<br>
※srcディレクトリにビルド前のコード、distディレクトリにビルド後のコードが入っています。

## 開発環境
- macOS
- VS Code
- webpack

## 使用言語
- HTML
- SCSS
- JavaScript
